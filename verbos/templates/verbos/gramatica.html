Subjonctif dans la subordonnée
La subordonnée est au subjontif si
  1. On exprime un opinion sur une forme négative :
    • no creo que venga
  2. avec des verbes exprimant le regret ,le souhait , le doute , :
    • souhait : deseo que los chicos se divertan
    • désir : exigimos que el Presidente nos reciba
  3. apres une expression qui marque le but, l'obligation :
    • es necessario que vengas
    • necesitamos que construyan nuevas industrias en la zona
  4. apres AUNQUE quand il est suivi d'un fait éventuel : aunque me lo repetieras , no te lo creeria
  5. apres ANTES DE QUE
    • El gobierno debe actuar antes de que sea demasiado tarde
  6. apres CUANDO pour parler d'un fait non accompli :
    • cuando llegues , iremos a buscarte
    • las energias altenativas se desarollaran cuando se acaben las reservas de petroléo
Ningun o ninguno ?
  1. Ninguno si pas de nom après
  2. ningun si il y a un nom masculin / mais ninguna si feminin
  3. ex ningun pico mais no hay ninguno , hay ninguna tienda , no hay ninguna
Duracion
  1. Hace + cantidad de tiempo + que + verbo hace mas que tres anos que vivo en espana
  2. Desde hace + cantidad de tiempo no veo Carlos desde hace un ano
  3. Desde + fecha no veo Carlos desde enero
  4. Desde que + verbo esta en grenada desde que esmpezo el curso

Pronoms pour COD / COI
Rappel
  • le COD repond à la question " qué ?" Astrid come un melon " / que come Astrid ? un melon = COD
  • le COI repond à la question "a quien ?" Astyrid regala un tiburon a Brigitte . A quien Astrid regala un tiburon ? A Brigitte = COI
Pronumbre COD COI
yo me me
tu te te
el / ella lo / la le
nosotros os os
vosotros os os
ellos / ellas los las les
exemplos
  • Astrid lo regala ( el tiburon COD )
  • Astrid me regala un tiburon ( COI )
  • Astrid le regala un tiburon a Brigitte ( on repéte toujours le COI ...)
  • Astrid se lo regala a Brigitte
    Contexte avec double pronom COD et COI dans la même phrase
  1. le pronom du COI pour les troisiemes personnes devient "se" en place de "le/les" . ex : tengo que decirselo
  2. format en fonction des conjuguaisons :
    • lorsque le verbe est conjugué : "COI + COD + verbe" . ex Astrid se lo regala a Brigitte ..
    • lorsque le verbe est non conjugué ( infinitif / gerondif / imperatif) : "verbe + COI + COD (pegado)". ex : tengo que decirtelo
Imperatif
L'impératif se deduit très exactement du subjonctif présent du verbe avec l'exception suivante :
  • pour l'impératif affirmatif aux 2 ieme personnes ( tu / vosotros )
    1. pour le pluriel ( vosotros ) : l'imperatif est construit à partir de l'infinitif en remplacant le r final par un d : vivir -> vivid , ser -> sed .. Stop.
      • => sauf pour les formes pronominales
        • le d disparait : ex conciensciaos, despertaos
        • un accent s'invite la pluspart du temps
    2. pour le singulier ( tu ) :
      1. on reprends le subjontif présent / forme tu : ex cantar -> cantes ...
      2. ... on enleve le s final : cantes -> cante ...
      3. ... on remplace la derniere lettre ...
        • ... pour le premier groupe : le e devient a : cante-> canta
        • ... pour les 2ieme et 3ieme groupe : le a devient e : beber-> bebas -> beba -> bebe
